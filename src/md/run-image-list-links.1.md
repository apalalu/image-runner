# NAME

run-image-list-links - list symbolic links that point to run-image.

# SYNOPSIS

**run-image-list-links** [binDirectory]

# DESCRIPTION

**run-image-list-links** list symbolic links that point to run-image.\
if binDirectory, it will be defaulted to **$HOME/.bin**.
