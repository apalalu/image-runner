# NAME

run-image-sample-config - print a sample config file.

# SYNOPSIS

**run-image-sample-config**

# DESCRIPTION

**run-image-sample-config** print a sample config file.\
User can save the output into **~/.image-runner/config.yaml** as a starting point.
